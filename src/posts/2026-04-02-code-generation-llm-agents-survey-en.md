# Code Generation with LLM-based Agents: A Survey

**Paper:** A Survey on Code Generation with LLM-based Agents
**arXiv:** [2508.00083](https://arxiv.org/abs/2508.00083)
**Venue:** arXiv preprint (July 2025)

---

## Three Core Features

1. **Autonomy:** Task decomposition → coding → debugging → testing independently
2. **Expanded Scope:** From code snippets to full repository development, CI/CD, documentation, code review
3. **Engineering Practicality:** Focus shifting from algorithmic innovation to reliability, process management, tool integration

## Architecture Taxonomy

- **Single-agent:** ReAct, Self-Debugging, Tool-Augmented
- **Multi-agent:** Role-based (Architect/Developer/Tester/Reviewer), debate-based, hierarchical

## SDLC Coverage

Requirements analysis, design, implementation, testing, debugging, maintenance, deployment — all covered by current agent systems.

## Representative Tools

ChatDev, MetaGPT, Devin, Cursor, Claude Code, GitHub Copilot Workspace, SWE-Agent.
